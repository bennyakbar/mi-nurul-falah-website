# Technical & UI Specification

### 1️⃣ UI / UX

- Warna: Hijau & Kuning
- Font: Poppins / Gotham
- Layout sederhana
- Mudah dibaca orang tua

---

### 2️⃣ Frontend Rules

- Semantic HTML
- CSS Flexbox / Grid
- JS untuk:
    - Navbar
    - Slider
    - Form validation

---

### 3️⃣ Responsiveness

- Desktop ≥ 1024px
- Tablet 768–1023px
- Mobile ≤ 767px

---

### 4️⃣ Security

- Tidak ada login
- Tidak menyimpan data pengguna
- Form kontak non-persisten

### 5️⃣ UI / UX – PPDB

- Layout vertikal & informatif
- Ikon checklist untuk persyaratan
- Timeline visual untuk alur PPDB
- Tombol CTA jelas: **“Daftar Sekarang”**

---

### 6️⃣ Konten Halaman PPDB

**Struktur konten `ppdb.html`:**

1. Hero PPDB (Tahun Ajaran)
2. Sambutan singkat PPDB
3. Jadwal & Gelombang
4. Persyaratan Pendaftaran
5. Alur Pendaftaran
6. Biaya Pendidikan *(opsional)*
7. Tombol Pendaftaran
8. Kontak Panitia

---

### 7️⃣ Interaksi JavaScript (Opsional)

- Scroll to section
- Accordion persyaratan
- Highlight CTA
