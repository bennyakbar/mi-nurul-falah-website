# Panduan SEO Lokal & Checklist Peluncuran - MI Nurul Falah

Dokumen ini berisi strategi SEO Lokal yang telah dan perlu diterapkan untuk meningkatkan visibilitas website di mesin pencari (Google), khususnya untuk pencarian di wilayah Margahayu dan Kabupaten Bandung.

## 1. Riset Keyword Lokal
Target kata kunci yang digunakan dalam website ini:

| Tipe Keyword | Keyword Utama | Keyword Spesifik (Long-tail) |
| :--- | :--- | :--- |
| **Brand** | `MI Nurul Falah` | `Alamat MI Nurul Falah Sayati` |
| **Lokasi** | `Madrasah Ibtidaiyah Bandung` | `SD Islam di Margahayu`, `Sekolah dekat Sayati` |
| **Program** | `Sekolah Tahfidz Bandung` | `Program Tahfidz Margahayu` |
| **PPDB** | `PPDB MI Bandung` | `Biaya masuk MI Nurul Falah` |

**Strategi:** Gunakan variasi kata kunci ini saat menulis berita, artikel, atau update konten di masa depan.

---

## 2. Struktur Schema Markup (JSON-LD)
Website ini sekarang menggunakan **Schema.org Type: School**.
Kode ini membantu Google menampilkan info "Rich Snippet" seperti jam buka, alamat, dan nomor telepon di hasil pencarian.

Kode ini telah ditanam di:
- `index.html` (Homepage)
- `kontak.html` (Halaman Kontak)

## 3. Checklist Sebelum Publish (Go Live)
Lakukan pengecekan ini saat konten sudah final dan website siap diupload ke hosting publik.

### A. Konten & Meta Data
- [x] **Title Tags**: Sudah dioptimasi mengandung "Margahayu" dan "Bandung".
- [x] **Meta Descriptions**: Sudah mengandung Call-to-Action (CTA) dan keyword.
- [ ] **Proofreading**: Cek ulang ejaan nama jalan, nama guru, dan kontak.
- [ ] **Gambar**: Pastikan semua gambar baru yang diupload memiliki nama file yang jelas (contoh: `kegiatan-pramuka-mi-nurul-falah.jpg` BUKAN `IMG_1234.jpg`).

### B. Google Business Profile (Sangat Penting!)
Langkah ini dilakukan di luar website (di Google Maps).
1. Buka [Google Business Profile](https://www.google.com/business/).
2. Klaim listing "MI Nurul Falah".
3. Pastikan **Nama, Alamat, dan Nomor Telepon (NAP)** persis sama dengan yang ada di website (Footer & Halaman Kontak).
4. Tambahkan link website `https://minurulfalah.sch.id` ke profil Google Maps tersebut.

### C. Technical Check
- [ ] **Mobile Test**: Buka website di HP, pastikan tulisan terbaca jelas.
- [ ] **Links**: Klik semua menu dan link, pastikan tidak ada yang "404 Not Found".
- [ ] **Speed**: Resize gambar yang terlalu besar (di atas 200KB) sebelum upload.

### D. Analytics
- [ ] Pasang **Google Analytics** (GA4) untuk memantau pengunjung.
- [ ] Pasang **Google Search Console** untuk memantau performa keyword di Google.
